# ECC (ECOS Chip Compiler)

ECC 是一个基于 **Tauri + Vue 3 + TypeScript** 构建的高性能桌面端绘图与 AI 交互应用程序。它结合了 Rust 的原生性能和现代前端框架的灵活性，旨在为工程师提供流畅的版图设计与 AI 辅助分析体验。

## 🚀 技术栈

- **桌面框架**: [Tauri 1.5](https://tauri.app/) (Rust 后端)
- **前端框架**: [Vue 3](https://vuejs.org/) (Composition API)
- **图形引擎**: [PixiJS 8](https://pixijs.com/) (高性能 WebGL/WebGPU 渲染)
- **UI 组件**: [PrimeVue 4](https://primevue.org/) (Aura 主题)
- **样式方案**: [Tailwind CSS v4](https://tailwindcss.com/)
- **图标库**: [Remixicon](https://remixicon.com/) & [PrimeIcons](https://primevue.org/icons)
- **构建工具**: [Vite 7](https://vitejs.dev/)

## 🛠️ 环境准备

在开始开发之前，请确保您的系统已安装以下环境：

### 1. 安装 Node.js 和 pnpm
- 推荐使用 [Node.js](https://nodejs.org/) (LTS 版本)
- 安装 [pnpm](https://pnpm.io/): `npm install -g pnpm`

### 2. 安装 Rust (Tauri 必备)
Tauri 需要 Rust 环境来编译后端代码。

- **使用官方安装脚本 (macOS/Linux)**:
  ```bash
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  ```
- **Windows**:
  请从 [rustup.rs](https://rustup.rs/) 下载 `rustup-init.exe` 并运行。

安装完成后，运行 `rustc --version` 验证。

### 3. 系统依赖 (针对 Tauri) —— ⚠️ 必需
Tauri 需要调用操作系统的原生库。**如果不安装这些依赖，项目将无法编译和运行。**

- **macOS**:
  - 安装 Xcode Command Line Tools: `xcode-select --install`
- **Windows**:
  - 安装 [Visual Studio Build Tools 2022](https://visualstudio.microsoft.com/visual-cpp-build-tools/)，并勾选 "使用 C++ 的桌面开发"。
  - 安装 [WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) (Windows 10/11 通常已内置)。
- **Linux (Ubuntu/Debian)**:
  ```bash
  sudo apt update
  sudo apt install libwebkit2gtk-4.0-dev build-essential curl wget file libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev
  ```

## 📦 快速开始

### 安装项目依赖
```bash
pnpm install
```

### 开发调试
```bash
# 同时运行前端和 Tauri 后端 (推荐)
pnpm run tauri:dev

# 仅在浏览器中运行前端预览
pnpm run dev
```

### 项目构建
```bash
# 生成对应平台的安装包 (dmg, exe, deb等)
pnpm run tauri:build
```


## 💡 开发指南

### 使用组件 (PrimeVue)
本项目使用 PrimeVue 4。示例：
```vue
<script setup lang="ts">
import Button from 'primevue/button'
</script>

<template>
  <Button label="确认" icon="pi pi-check" />
</template>
```

### 使用图标
- **PrimeIcons**: 使用 `pi pi-*` 类名 (如 `<i class="pi pi-search"></i>`)
- **Remixicon**: 使用 `ri-*` 类名 (如 `<i class="ri-settings-line"></i>`)

### 状态管理
- 使用 **Pinia** 进行全局状态管理 (见 `src/stores/`)。
- 业务逻辑封装在 `src/composables/` 下。
